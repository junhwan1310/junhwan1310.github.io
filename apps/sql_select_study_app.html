SQL SELECT 학습 앱

📌 반드시 알아야 할 핵심 개념 (초보자 필수)

❗ WHERE에서는 집계함수 사용 불가
→ WHERE AVG(weight) >= 5 ❌ (안됨)

❗ GROUP BY는 "~별" 계산할 때 사용
→ 품종별, 배우별 같은 상황

❗ HAVING은 GROUP BY 이후 필터 → HAVING은 거의 항상 GROUP BY와 같이 사용

❗ "가장 큰 값" 찾기 공식 → ORDER BY DESC + LIMIT 1

❗ WITH 문법 핵심 → WITH 이름 AS ( ... ) 형태 필수
→ WITH 안에서는 완성된 결과표를 만들어야 함

아이콘 하나만 눌러도 SELECT 문제를 모아 보고, 유형별로 다시 복습하고, 정답 SQL과 핵심 포인트까지 한 번에 확인할 수 있게 만든 간단한 학습 앱입니다.

커리큘럼 순서
이미지에 나온 목차처럼 큰 개념 기준으로 먼저 보고, 그 안에서 기존 문제들을 필터링해서 보게 바꿨다.
기존 문제는 지우지 않고 그대로 둔 상태다.
문제 읽을 때 바로 찾을 것
1) 어떤 컬럼을 보여줄까? → SELECT
2) 어떤 행만 고를까? → WHERE
3) 정렬이 필요한가? → ORDER BY
4) 개수/평균/합계/최대/최소인가? → COUNT / AVG / SUM / MAX / MIN

빠른 반응 키워드
“수” → COUNT
“평균” → AVG
“합계” → SUM
“최대” → MAX
“최소” → MIN

NULL 정리
값 바꾸기 → IFNULL
행 제거 → IS NOT NULL

JOIN 빠른 구분
둘 다 있는 데이터 → JOIN
한쪽만 남은 데이터 찾기 → LEFT JOIN + IS NULL

GROUP BY / HAVING / WITH
그룹 나누기 → GROUP BY
그룹 결과 조건 → HAVING
중간 결과 이름 붙이기 → WITH
